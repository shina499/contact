# پروژه صفحه مخاطبین آنلاین

این پروژه یک صفحه ساده برای مدیریت مخاطبین آنلاین است که به کاربران اجازه می‌دهد تا مخاطبین خود را اضافه، ویرایش و حذف کنند. این صفحه با استفاده از HTML5، Vanilla JavaScript و Tailwind CSS ساخته شده است و به راحتی می‌توان از آن برای ذخیره و مدیریت اطلاعات مخاطبین استفاده کرد.

## ویژگی‌ها

- اضافه کردن مخاطب: کاربران می‌توانند اطلاعات جدید یک مخاطب را وارد کرده و آن را به لیست اضافه کنند.
- ویرایش مخاطب: اطلاعات هر مخاطب قابل ویرایش است و می‌توان تغییرات را در آن اعمال کرد.
- حذف مخاطب: کاربران می‌توانند مخاطبین خود را حذف کنند.
- رابط کاربری ساده و واکنش‌گرا: طراحی صفحه به گونه‌ای است که به راحتی می‌توان در دستگاه‌های مختلف از آن استفاده کرد.

## تکنولوژی‌های استفاده شده

- HTML5: برای ساختار صفحه و فرم‌های ورودی استفاده شده است.
- Vanilla JavaScript: برای مدیریت تعاملات کاربر و پردازش داده‌ها از JavaScript خالص (بدون استفاده از فریمورک‌های اضافی) استفاده شده است.
- Tailwind CSS: برای طراحی سریع و واکنش‌گرا از این فریم‌ورک CSS استفاده شده است تا ظاهر و احساس طراحی زیبا و کاربرپسند باشد.

## نحوه استفاده💻

1. ابتدا فایل‌های پروژه را بر روی سیستم خود دانلود یا کلون کنید.
2. به پوشه پروژه بروید.
3. فایل index.html را در مرورگر خود باز کنید.
4. می‌توانید مخاطبین خود را با استفاده از فرم موجود در صفحه اضافه کنید.
5. برای ویرایش یا حذف یک مخاطب، بر روی دکمه‌های مربوطه کلیک کنید.
